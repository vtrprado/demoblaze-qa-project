# QA Project – Demoblaze
Este repositório contém um projeto simples de testes funcionais manuais realizado no website Demoblaze, como exercício prático de QA.

## Objetivo
Validar as funcionalidades principais do e-commerce Demoblaze por meio de testes manuais estruturados.

## Conteúdo do Projeto
- Plano de Teste
- Casos de Teste (Test Cases)
- Relatórios de Bugs (Bug Reports)
- Evidências (prints da execução) 

## Técnicas Aplicadas
- Teste funcional manual  
- Teste exploratório  

## Ferramentas Utilizadas
- Excel  
- Google Chrome  
- GitHub
